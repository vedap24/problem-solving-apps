
# 💡 DSA 5-Step Logic Builder

A smart AI mentor to help you build Data Structures & Algorithms intuition without giving away the final code\!

**Live App:** [https://dsa-mentor.streamlit.app](https://dsa-mentor.streamlit.app)

## 🚀 What is this?

Getting stuck on LeetCode is normal, but looking directly at the solution ruins the learning process. I built this app using the Gemini API to act as your personal DSA Mentor.

You paste a problem description or title, and it generates a 5-step logic roadmap. The best part? **Zero code spoilers.** It strictly provides hints and pseudo-logic, forcing you to build the intuition and write the code yourself\!

## ✨ Features

  * **5-Step Blueprint:** Breaks down any problem into:
    1.  Core Essence
    2.  Brute Force Flaws
    3.  Pattern Recognition
    4.  Logic Blueprint
    5.  Complexity
  * **No Spoilers:** Prevents you from cheating yourself by withholding the final code solution.
  * **Bring Your Own Key:** Securely enter your free Gemini API key in the sidebar to get started.

## 🛠️ Tech Stack

  * **Language:** Python
  * **Frontend:** Streamlit
  * **AI Model:** Google Gemini 2.5 Flash (`google-generativeai`)

## 💻 How to Run Locally

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/vedap24/problem-solving-apps.git
    cd problem-solving-apps
    ```

2.  **Install the required libraries:**

    ```bash
    pip install streamlit google-generativeai
    ```

3.  **Run the Streamlit app:**

    ```bash
    streamlit run app.py
    ```

4.  **Add your API Key:**
    Open the local URL provided in your terminal. Enter your free Gemini API key in the sidebar to start generating logic blueprints\!
